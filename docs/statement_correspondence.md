# Statement Correspondence

This note records how the main paper-level mathematical statements are exposed
as Lean declarations.

## Ambient Ring

The generic `m × n` matrix is represented in

```lean
MvPolynomial (Fin m × Fin n) k
```

The coordinate variable in row `i` and column `j` is
`MvPolynomial.X (i, j)`, and the generic matrix itself is Mathlib's
`Matrix.mvPolynomialX`.

## Generic Minors and Determinantal Ideals

| Mathematical object | Lean declaration |
| --- | --- |
| Row and column index data for a `t × t` minor | `Matrix.MinorIndex`, `Determinantal.MinorIndex` |
| Generic minor polynomial | `Matrix.MinorIndex.mvPolynomialMinor`, `Determinantal.genericMinor` |
| Set of all `t × t` generic minors | `Determinantal.minorSet` |
| Ideal generated by all `t × t` minors | `MvPolynomial.determinantalIdeal`, `Determinantal.detIdeal` |
| The paper's `J_r` | `Determinantal.Jr` |
| The paper's `G_{r+1}` | `Determinantal.GrPlusOne` |

## Term Orders and Leading Terms

| Mathematical statement | Lean declaration |
